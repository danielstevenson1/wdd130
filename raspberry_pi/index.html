Flash sd card

Flashing an OS onto a sd card is the first thing you need to do to start using your pi.

  • Download raspberry pi Imager from https://www.raspberrypi.com/software/
  • insert sd card into pc with adapter and choose your OS then click flash!
  • when you click flash make sure you have chosen the corect storage device to flash to.

Start your pi

once you have flashed your os to your sd chard plug it into the back of your pi.

  • power on you raspberry pi and wait for it to boot.
  • Depending on your os the set up prossess will be diffrent but for the most part it will be simple.
  • go through the steps of sutting up your pi and make sure you put in all the corect information.

terminal

once your pi is set up we will need to update it.

  • Either click on the terminal ccomand tor type ctrl + alt+ t
  • Once the terminal is open type sudo apt update then hit enter.
  • then once the operation is done type sudo apt full-upgrade -y to finish off the update

Apps

Lets install some apps to get started

  • there are many ways to install apps through linux but the comand line works the best.
  • type sudo apt install "options below"
  • gimp, inkscape, timeshift, steam, vlc and much more

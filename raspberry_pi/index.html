SD Card

How to install linux on a sd card so we start using our Raspberry PI:

  • Install Raspberry Pi imager from this link (https://www.raspberrypi.com/software/)
  • insrt your sd card using an adapter then open the flasher and flash your desired OS
  • Once you have the os on your sd card put it in your Pi and plug in the power and wait for it to boot

Terminal

Updating your Pi

  • Once you have set up your pi we need to make sure it is up to date.
  • Pres ctrl+alt+t to open the terminal and type sudo apt update
  • one that is done type sudo apt upgrade -y
  • Once your pi is up to date you can start experimenting

Installing open sourse software on your pi

  • there are many diffrent ways to install diffrent applications but we wull use the apt library
  • By using "sudo apt install (application) we can install most aplications"

SD Card

How to install Linux on an SD card so we can start using our Raspberry Pi:

  • Install Raspberry Pi Imager from this link (https://www.raspberrypi.com/software/)
  • Insert your SD card using an adapter then open the flasher and flash your desired OS
  • Once you have the OS on your SD card, put it in your Pi and plug in the power and wait for it to boot

Terminal

Updating your Pi

  • Once you have set up your Pi, we need to make sure it is up to date.
  • Press Ctrl+Alt+T to open the terminal and type sudo apt update
  • Once that is done, type sudo apt upgrade -y
  • Once your Pi is up to date, you can start experimenting

Install Apps

Installing open-source software on your Pi

  • There are many different ways to install different applications, but we will use the apt library
  • By using sudo apt install (application), we can install most applications
  • Here is a list of applications you can install using this process: GIMP, Inkscape, Blender, Zoom, SteamLink, LibreOffice, and much more
